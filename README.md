## Git Commits
커밋 메시지에 추가되는 간단한 규칙

```shell
<type>: <description>

ex) refactor: jpa -> mybatis 변경 작업
```
* feat: - 새로운 기능 추가와 관련된 커밋 
* fix: - 버그 수정과 관련된 커밋
* chore: - 일상적이고 미세한 작업을 나타내는 커밋
* refactor: - 코드 리팩토링과 관련된 커밋
* perf: - 성능 개선과 관련된 커밋
* test: - 테스트 관련 작업과 관련된 커밋
